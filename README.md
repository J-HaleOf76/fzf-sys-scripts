| :------------ | :---------------------------------------------------------- |
| fzf_pacman.sh | manage packages with pacman and aur helper (paru, yay, ...) |
| fzf_man.sh    | search and open man pages                                   |
| fzf_trash.sh  | manage files/folders with trash-cli                         |
